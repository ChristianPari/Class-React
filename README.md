# MERN User System Boilerplate
This is an in-class assigned project that is being used to teach how to use React! This is partially done in group coding but mostly of on own time coding along with indivudual research and implementation of new concepts.
## Goal
To setup a decently expanded upon layout for a User System Front End using React! Simple Home for User to Login or Sign Up, a Login page and a Sign Up page, as well as a 404 page for when a user attempts to reach a paht of our server that does not exist.
